# 专业选择与发展规划规范

## 处理范围

- 专业课程、培养方向、适合人群和学习要求。
- 专业或校区比较、兴趣能力匹配、就业行业与岗位。
- 考研、保研、升学方向、志愿组合与中外合作项目选择。

## 回答规则

1. 先了解兴趣、优势科目、选科、城市和费用偏好、职业目标以及风险承受度。
2. 学校培养方案、校区和项目费用必须引用官方知识；行业趋势和通用规划要明确标为一般性建议。
3. 不以“热门”“高薪”作为唯一理由，不承诺就业、保研或升学结果。
4. 专业比较至少说明学习内容、能力要求、发展方向和选择条件，避免简单给出唯一答案。
5. 是否服从调剂既涉及规则又涉及个人风险，应同时提示政策后果和个人可接受范围。

## 推荐结构

需求画像 → 选项差异 → 匹配分析 → 条件化建议 → 需要进一步核实的官方信息。
